# PEDANT - Cosmetics E-Commerce Platform

🌐 **[View Live Website](https://tamarazriel.github.io/PEDANT-website/)**

## Overview
PEDANT is an e-commerce platform designed for selling professional cosmetics and skincare brands. The project was developed as part of the Computer Science studies at the Jerusalem College of Technology (Machon Tal). It simulates a fully functional virtual storefront, featuring a product catalog, user authentication, a shopping cart, and a checkout process.

## Key Features
* **User Management:** Registration, login, and personalized user profile pages.
* **Product Catalog:** Dedicated pages for different cosmetic brands with dynamic product displays.
* **Shopping Cart:** Add-to-cart functionality, real-time quantity updates, and total price calculation.
* **Checkout Simulation:** End-to-end simulated checkout process.
* **Automated Testing:** E2E testing scripts implemented in Python to ensure site functionality.

## Technology Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Testing:** Python

## Installation & Usage
1. Clone the repository:
   ```bash
   git clone [https://github.com/TamarAzriel/PEDANT-website.git](https://github.com/TamarAzriel/PEDANT-website.git)
   ```
2. Open `index.html` in your preferred web browser.
3. To run the automated tests, ensure Python is installed and execute:
   ```bash
   python tests.py
   ```

## Project Structure
* `index.html` - Main landing page.
* `css/` - Stylesheets (`styles.css`, `luxury.css`).
* `JavaScript/` - Client-side logic (`auth.js`, `cart.js`, `products.js`, etc.).
* `images/` - Static assets and product images.
* `tests.py` - Automated test suite.
